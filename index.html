Facial Emotion Recognition

The CNN will classify the image into one of the seven emotional states namely - angry, disgusted, fearful, happy, neutral, sad and surprised. VGG16 pre-trained model is fine-tuned The model takes images of dimension (50, 50, 3) and the output is one of the seven emotion classes. The base model(i.e VGG16 model) layers is set to non-trainable.

Rain Prediction Application

The project aims to create Decision Tree models that can predict if it will rain tomorrow and today based on a set of features like the Maximum temperature, Evaporation, Sunshine, Wind Gust Speed, Humidity at 9 am etc. The web application is created and rendered using Flask.
